  1. id 属性标识符（用于引用）,不应依赖其语义处理相应元素

  2. title 属性：

    • 链接：描述目标信息
    • 图片： 版权/描述
    • 引用： 来源信息
    • 交互元素：操作指南
    • ….
  1. link 元素，元数据，用来描述文档本身与其他资源的关系；必须包含 rel 以及 href 属性。

link + rel + author, link + rel + license 都有预定义的语义
