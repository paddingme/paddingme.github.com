

CSS中一些标签不太熟悉，记录在这里。

* q标签的定义和用法：
>`<q>`标签定义极短的行内引用,可选属性`cite`，来定义引用的出处或者来源。
 例如：
 <q cite="padding.me/">这是一个行内短引用，cite指明出处</q>
 浏览器显示：
 ><q cite="padding.me/">这是一个行内短引用，cite指明出处</q>

 *cite还可以用于指出引言，标注出处或者来源。
 例如：
     <cite>鲁迅</cite>先生说：<q>地上本来没有路，走得人多了，便有了路</q> 

 浏览器会显示：
 ><cite>鲁迅</cite>先生说：<q>地上本来没有路，走得人多了，便有了路</q>
